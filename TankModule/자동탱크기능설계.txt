물체 자동 이동 프로그램 설계

①②③④⑤⑥⑦⑧⑨⑩⑪⑫⑬⑭⑮⑯⑰⑱⑲⑳

 1. 기능

   1.1 아듀이노 제어
   
        1) 대기모드 ON
           ⇒ 동작을 멈추고 최소한의 전압으로 대기하는 기능.
              ① 동작 모드가 수동 일 경우
                 → 라즈베리 파이로부터 신호가 전송되었을경우
                    대기 모드로 전환되며 최소 전압으로 대기하는것으로 함.
              ② 동작 모드가 자동 일 경우
                 → 해당사항 없음

        2) 대기모드 OFF
           ⇒ 대기모드에서 동작모드로 전환하는 기능
              ① 동작 모드가 수동 일 경우
                 → 라즈베리 파이로부터 신호가 전송되었을경우 동작 모드로 전환
              ② 동작 모드가 자동 일 경우
                 → 해당사항 없음

        3) 자동모드
           ⇒ 물체 자체의 거리 센서를 이용하여 장애물을 파악하고 장애물을 피해
              자동으로 이동하는 기능.

        4) 수동모드
           ⇒ 신호를 받아서 제어하는 기능으로서
              이동처리(전진, 후진, 회전, 속도전환, 정지등)와
              전원제어및, 모드전환 기능

        5) 모드전환
           ⇒ 신호에 의한 자동/수동모드、대기모드ON/OFF 전환기능.

        6) 전진이동
           ⇒ 물체를 전면방향으로 이동 시키는 기능 
              (※ 전면이란 거리감지 센서가 있는 부분을 기준으로 함.)

        7) 후진이동
           ⇒ 물체를 후면방향으로 이동 시키는 기능 
              (※ 후면이란 거리감지 센서가 있는 부분의 뒷부분을 말함.)

        8) 좌측회전
           ⇒ 물체를 좌측방향으로 회전 시키는 기능 
              (※ 전면방형을 기준으로 좌측회전.)

        9) 우측회전
           ⇒ 물체를 우측방향으로 회전 시키는 기능 
              (※ 전면방향을 기준으로 우측회전.)

       10) 속도전환
           ⇒ 물체의 속도를 전환하는 기능으로 다음의 두 모드에 대한 처리기능.
              (※ 전진이동일때만 해당.)
              ① 동작 모드가 수동 일 경우
                 신호에 의한 속도 전환(1:저속, 2:중속, 3:고속)
              ② 동작 모드가 자동 일 경우
                 전면의 장애물 유무에 의해 자동 속도 전환
                 (50Cm이하:저속, 50Cm-100Cm:중속, 100Cm이상:고속)가 필요

       11) 정지
           ⇒ 물체를 정지시키는 기능.
              (※ 대기모드와는 틀림.)

       12) 장해물 감지
           ⇒ 전면에 장애물이 있는지를 감지하는 기능.
              (※ 자동운전모드 의 경우에만 해당.)

       13) LED 표시
           ⇒ 물체의 동작/대기 및 수동/자동 모드의 내용을 LED램프를 사용해서 알수있도록 하는 기능.
              (※ 예: 대기모드⇒LED OFF、수동모드⇒LED ON、자동모드⇒LED 점멸)


   1.2 라즈베리파이 제어

        1) 카메라동작제어(ON, OFF)
           ⇒ 카메라의 설정및 동작등을 제어하는 기능

        2) 카메라 녹화
           ⇒ 카메라 동작시에 카메라로부터 들어온 영상을 녹화
              또는, 사진촬영 하는 기능

        3) 카메라 영상 스트림
           ⇒ 카메라 작동시에 카메라로부터 들어온 영상을 스트림해서 보내주는 기능

        4) 카메라 각도 조정
           ⇒ 카메라를 좌우로 이동시키는 기능


   1.3  원격제어 조정

        1) 라즈베리 파이 에 정보 송신기능
           ⇒ 카메라의 설정및 동작등을 제어하는 기능

        2) 동영상 표시기능
           ⇒ 라즈베리파이로부터 들어오는 영상을 단말에 표시하는 기능
